⚙️ API Quick Reference

All endpoints available via REST API. No authentication required.

GET/api/pfds?lat=25.76&lon=-80.19&return_period_yr=100&duration_hr=24
NOAA Atlas 14 PFDS depth for a single duration and return period at a coordinate.
GET/api/pfds/table?lat=25.76&lon=-80.19
Full IDF table — all durations x all return periods at a coordinate.
GET/api/runoff?cn=75&rainfall_in=8.5&ia_ratio=0.2
NRCS TR-55 runoff depth. Returns Q, S, Ia, runoff fraction, volume.
POST/api/run
Full rainfall + DEM depth-grid analysis. JSON body: lat, lon, dem_path, return_period_yr, duration_hr, curve_number.
