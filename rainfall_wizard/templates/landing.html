Core Capabilities

NOAA PFDS Rainfall Analysis for Engineers

The CVG Rainfall Wizard fetches official NOAA Precipitation Frequency Data Server (PFDS) depths for any latitude/longitude in the U.S. and immediately computes runoff volumes using NRCS Curve Number methodology.

🌧️

NOAA PFDS Rainfall Depths

Official precip frequency estimates from NOAA Atlas 14. Supports all standard return periods (2-yr through 1,000-yr) and storm durations (5-min through 60-day) for any CONUS location.

📐

NRCS TR-55 Runoff (CN Method)

Curve Number based runoff depth and volume computation per USDA NRCS Technical Release 55. Supply your drainage area CN and get runoff depth, volume, and peak flow estimate instantly.

📊

Full IDF Table Output

Auto-generate complete Intensity-Duration-Frequency tables for your location — covering all standard return periods and durations. Ready for stormwater design reports in seconds.

🌊

SLR Climate Adjustment

Optionally apply a sea level rise scenario for future-conditions analysis — integrates directly with CVG's SLR Wizard projections for combined flood risk studies.

⚡

Instant Results

Enter coordinates, return period, and storm duration — results in under 2 seconds. No GIS software, no local data downloads, no configuration required.

🌐

REST API for Automation

Full OpenAPI integration for batch queries across large project portfolios. Automate IDF table generation for hundreds of sites in a single workflow.
