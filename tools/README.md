# CVG Rainfall Wizard — Tools

> © Clearview Geographic LLC | Internal Use Only

Utility scripts for the CVG Rainfall Wizard.

## Planned Tools

- `cache_pfds_bulk.py` — Pre-cache Atlas 14 data for a county/state bounding box
- `validate_dem.py` — DEM integrity checker (nodata %, unit detection)
- `cn_raster_from_nlcd.py` — Build spatially varying CN raster from NLCD + SSURGO HSG
- `batch_report_gen.py` — Re-generate reports from existing JSON output files
- `climate_adjust_pfds.py` — Apply climate change multipliers to Atlas 14 depths
